# 🛒 Walmart Sales Data Analysis (SQL + Python)

A data analytics project focused on extracting business insights from Walmart sales data using **SQL and Python**.  
This project demonstrates **data querying, business problem solving, and analytical thinking** by answering real-world retail questions such as revenue trends, payment behavior, customer ratings, and branch performance.

---

# 📊 Project Overview

Retail companies generate massive transactional datasets. Analyzing this data helps organizations understand **customer behavior, revenue trends, and operational performance**.

This project performs **exploratory data analysis and business analytics** on Walmart sales data to identify patterns such as:

- Most used payment methods
- Best performing product categories
- Branch performance trends
- Revenue decline analysis
- Customer rating insights

The analysis is done using **PostgreSQL queries and Python-based exploration** to simulate real-world **data analyst workflows.**

---

# 🎯 Business Problems Solved

The project answers several **real business questions**, including:

### 1️⃣ Payment Behavior Analysis
- Identify different **payment methods used by customers**
- Calculate **number of transactions and quantity sold per method**

### 2️⃣ Best Rated Product Categories
- Determine the **highest rated category in each branch**
- Uses **window functions (RANK)** for ranking analysis

### 3️⃣ Branch Transaction Analysis
- Identify the **busiest day for each Walmart branch**
- Helps understand **peak sales periods**

### 4️⃣ Sales by Payment Method
- Calculate **total quantity sold per payment type**

### 5️⃣ Customer Satisfaction Insights
- Analyze **average, minimum, and maximum rating by category and city**

### 6️⃣ Profitability Analysis
- Calculate **total profit by category**

Profit Formula Used:

Profit = Unit Price × Quantity × Profit Margin

### 7️⃣ Preferred Payment Method per Branch
- Identify the **most commonly used payment method** in each branch

### 8️⃣ Sales Shift Analysis

Transactions categorized into:

- Morning  
- Afternoon  
- Evening  

Helps businesses optimize **staff scheduling and store operations.**

### 9️⃣ Revenue Decline Detection

- Identify **top 5 branches with highest revenue decline**
- Compare **2023 vs 2022 revenue performance**
